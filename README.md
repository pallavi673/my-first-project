# my-first-project
 Sales Data Analysis Mini Project

## Overview
This project analyzes a Superstore Sales dataset to extract key insights. 
It includes:
- Data cleaning and processing using Pandas
- Visualizations using Matplotlib and Seaborn
- Outputs saved as CSVs for further use

## Skills Used
Python, Pandas, Matplotlib, Seaborn, Data Analysis, Data Visualization

## Instructions
- Open `sales_analysis.ipynb` to view the notebook
- Dataset: `superstore_sales.csv`
- Outputs: `sales_by_region.csv`, `category_sales.csv`, optional plot images
